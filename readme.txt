Include images in your Wordpress RSS feed.

== Description ==

Simple plugin that allows to include the thumbnail image of your post in your Wordpress RSS & RSS2 feeds.
You can select the  size of the images on the admin panel.

Based on WPP RSS Images by Alain Gonzalez
http://web-argument.com/wp-rss-images-wordpress-plugin/

== Installation ==

1. Upload WP RSS Images folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings/WP RSS Images and select the image size (Thumbnail, Medium, Full) and feed type (RSS, RSS2).
